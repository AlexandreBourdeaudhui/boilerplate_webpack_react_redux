# :pick: Starter React & Redux with Webpack v3.6 :gear:


## :sunny: How that work ?
 * Download it.
 * Install Package :
  * With **Yarn** : `yarn`
  * With **NPM** : `npm install`


 * Launch _watcher_ :
   * With **Yarn** : `yarn start`
   * With **NPM** : `npm start`


 * Go http://localhost:3000
 * Start to Dev
 * Have fun ! :rocket:
